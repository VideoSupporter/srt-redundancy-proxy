# Privacy Policy for SRT Redundancy Proxy

**Last Updated: May 15, 2026**

## Introduction

This Privacy Policy describes how SRT Redundancy Proxy ("we", "our", or "the application") handles user information. We are committed to protecting your privacy and being transparent about the application's data practices.

## Information Collection

**SRT Redundancy Proxy does not collect, transmit, or store personal information for analytics, tracking, advertising, or profiling.**

The application operates on your local computer and does not:

- Collect personal data
- Track user behavior
- Send analytics or telemetry data
- Connect to our servers for data collection
- Share information with third parties

## Local Data Storage

SRT Redundancy Proxy may store the following data locally on your computer:

### Application Settings

- Input port
- Output destination host names or IP addresses
- Output destination ports
- Output enabled/disabled state
- Window position or UI state, when supported

### Logs

- Application start and stop events
- Proxy start and stop events
- SRT connection and disconnection events
- Output enable/disable events
- Error details needed for troubleshooting

This data is stored locally on your device and is not sent to us.

## Stream Data

The application receives an SRT stream from the input configured by the user and forwards it to the output destinations configured by the user.

- Stream data is transmitted only to the SRT destinations configured by the user.
- The application does not upload stream data to our servers.
- The application does not record stream data by itself unless a receiver or external tool saves the stream.
- Users are responsible for the destination addresses and for handling stream data according to their organization or local requirements.

## Network Activity

SRT Redundancy Proxy communicates by:

- Listening for a user-provided SRT input stream
- Connecting to user-configured SRT output destinations
- Sending the received SRT payload to enabled outputs

The application only communicates with the SRT endpoints configured by the user for relay operation.
No analytics, advertising, or tracking network connections are made by the application.

## Third-Party Services

SRT Redundancy Proxy does not integrate with or use third-party analytics, tracking, or advertising services.

The application uses third-party software components for its functionality, including SRT-related libraries and Windows platform components. These components are used locally as part of the application.

## Data Security

Since settings and logs are stored locally on your computer:

- You are responsible for the security of your device.
- Use firewall and network settings appropriate for your environment.
- Treat relayed stream data as sensitive when it contains private or confidential content.

The current version does not enable SRT encryption by default.

## Children's Privacy

SRT Redundancy Proxy does not knowingly collect any information from anyone, including children under the age of 13.

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the application documentation with an updated "Last Updated" date.
